Pembuktian (Laptop Fisik)

Anda menggunakan Strategi B. Saatnya membuktikan bahwa kabel "Bridged" berfungsi dan Laptop Anda bisa berkomunikasi dengan Server Virtual.

Misi Pengujian

Persiapan Hardware:

  • PC Server: Menjalankan VMware (VMnet0 = Bridged).
  • Laptop Client: Windows Asli.
  • Koneksi: Kabel LAN terpasang (Lampu nyala).
  • Setting: Wi-Fi Laptop harus OFF.

Target Checklist:

  • Mendapat IP 10.10.10.x.
  • Ping Gateway 10.10.10.1 (Reply).
  • Ping Google 8.8.8.8 (Reply/NAT).
  • Akses Web www.lks2026.id.
1

Koneksi Fisik & Network

Persiapan di Laptop Client.

1. Matikan Wi-Fi (Wajib)

Agar trafik internet tidak bingung lewat mana, matikan Wi-Fi Laptop sekarang.

2. Hubungkan Kabel LAN

Colokkan kabel ke Port LAN PC Server. Pastikan ikon Ethernet di Windows tidak silang merah.

3. Setting IP (DHCP)

Di Laptop Client:

  • Tekan Win + R, ketik ncpa.cpl.
  • Klik Kanan Ethernet > Properties > IPv4.
  • Pilih Obtain an IP address automatically.
  • Klik OK & Close.
2

Tes Koneksi (CMD Laptop)

Buka Command Prompt di Laptop Fisik.

1. Cek IP Address

Ketik di CMD Laptop:

C:\> ipconfig
Target: IPv4: 10.10.10.x | Gateway: 10.10.10.1

2. Ping Gateway & Internet

C:\> ping 10.10.10.1
C:\> ping 8.8.8.8
3

Tes Layanan (Browser)

Pengujian Web & DNS.

1. Tes Domain (Nslookup)

Pastikan Laptop mengenal domain kita:

C:\> nslookup www.lks2026.id

2. Buka Website

Buka Browser di Laptop, akses http://www.lks2026.id.

Jika Load Balancer Aktif (Fase 3/7)

"Ini Server WEB 01" / "WEB 02"

Sempurna

Jika Gagal Load

Jika DNS resolve tapi web tidak muncul, mungkin Nginx/HAProxy belum aktif di Router. Ini Wajar jika Anda belum masuk Fase 7.

Troubleshooting Fisik

Dapat IP tapi tidak bisa Ping
Firewall: Windows Firewall di Laptop sering memblokir ping. Coba matikan sebentar.
Routing: Cek di Router fw, apakah `ip_forward=1` sudah aktif?
Unidentified Network (Tidak dapat IP)
Bridging Salah: Masalah #1. Buka Virtual Network Editor di PC Server (Host). Pastikan VMnet0 diset ke Bridged dan pilih nama LAN Card Realtek/Intel (jangan Auto).
