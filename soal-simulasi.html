2. Konfigurasi Layanan (Services)

A. Domain Name System (DNS)

  • Nama Domain Utama: skilljateng.net
  • Install paket Bind9 pada Router (fw).
  • Buat Record A: www.skilljateng.net mengarah ke IP 10.10.50.1 (IP Router).
  • Buat Record A: mail.skilljateng.net mengarah ke IP 10.10.50.1 (IP Router).
  • Pastikan pengujian nslookup dari Client berhasil menampilkan IP tersebut.

B. Web Server & Load Balancer

  • Install Apache2 pada web01 dan web02.
  • Ubah port default HTTP menjadi HTTPS (Port 443) menggunakan sertifikat Self-Signed (OpenSSL).
  • Install HAProxy pada Router (fw). Konfigurasi agar trafik port 443 dibebankan (balancing) ke 10.10.50.10 dan 10.10.50.20.

C. Virtual IP & VPN (Tingkat Lanjut)

  • Virtual IP: Tambahkan IP sekunder 10.10.50.254 pada interface DMZ di Router (fw). IP ini harus bisa di-ping oleh web01 dan web02.
  • VPN Server: Install layanan VPN (PPTP atau Wireguard) di Router (fw).
    • IP Pool untuk Client VPN: Range 10.10.99.10 s.d. 10.10.99.20.
    • User VPN: peserta, Password: vpn123.
    • Client yang terhubung VPN harus bisa melakukan Ping ke Server Internal (192.168.100.10).

3. Keamanan Jaringan (Firewall)

Terapkan aturan Firewall menggunakan Iptables atau Nftables pada Router (fw):
