Cloning & Hardware (PENTING!)

[cite_start]

Menyiapkan "Kabel Fisik" ke Client[cite: 197].

1. Full Clone

Klik kanan Debian-Master > Manage > Clone. Pilih Full Clone. [cite_start]Nama VM: fw[cite: 199].

2. Network Adapter (Mode Bridged)

Atur 4 Adapter di VM Settings. [cite_start]Perhatikan Adapter 4[cite: 201]:

[cite_start] [cite_start] [cite_start] [cite_start]
No Setting Fungsi
1 NAT / BridgedInternet (ISP) [cite: 207]
2 VMnet10 (Custom)DMZ (Web/Mail) [cite: 210]
3 VMnet11 (Custom)Internal Server [cite: 213]
4 Bridged (LAN Card)Ke Laptop Fisik [cite: 216]

*Pastikan kabel LAN sudah tercolok antara PC Server dan Laptop Client.

2

Konfigurasi IP

[cite_start]

Nyalakan VM & Login Root[cite: 218].

1. Cek Interface

[cite_start]

Lihat nama asli (ens33/37/etc) - Klik kode untuk info[cite: 220]:

root@fw:~# ip link show

2. Edit File Interfaces

root@fw:~# nano /etc/network/interfaces

Hapus semua, Paste kode ini (Sesuaikan nama 'ens' Anda):

# Loopback auto lo iface lo inet loopback # INTERNET (Bridged/NAT) auto ens33 iface ens33 inet dhcp # DMZ (Web Server Gateway) auto ens37 iface ens37 inet static address 172.16.10.1/24 # INTERNAL (Mail/LDAP Gateway) auto ens38 iface ens38 inet static address 192.168.10.1/24 # CLIENT (Fisik via LAN Card) auto ens39 iface ens39 inet static address 10.10.10.1/24

3. Apply Config

root@fw:~# systemctl restart networking
root@fw:~# ip a
3

Routing & NAT

[cite_start]

Wajib agar Client bisa akses Web[cite: 245].

Persiapan DVD

[cite_start]

Masukkan DVD-1 & DVD-2 (Binary) ke VMware sebelum lanjut[cite: 249].

root@fw:~# apt-cdrom add root@fw:~# apt update

1. IP Forwarding

root@fw:~# nano /etc/sysctl.conf
[cite_start]

Hapus pagar (#) di baris[cite: 251]:

net.ipv4.ip_forward=1
root@fw:~# sysctl -p

2. NAT Masquerade

root@fw:~# apt install iptables iptables-persistent -y
[cite_start]

*Pilih YES saat ditanya simpan rule[cite: 254].

# Rule NAT ke ens33 (Internet)
root@fw:~# iptables -t nat -A POSTROUTING -o ens33 -j MASQUERADE
# Simpan Permanen
root@fw:~# netfilter-persistent save
4

DHCP Server

[cite_start]

Memberi IP otomatis ke Laptop Fisik[cite: 260].

1. Install & Set Interface

root@fw:~# apt install isc-dhcp-server -y root@fw:~# nano /etc/default/isc-dhcp-server
[cite_start]

Isi dengan interface arah Client (ens39)[cite: 267]:

INTERFACESv4="ens39"

2. Config Subnet

root@fw:~# nano /etc/dhcp/dhcpd.conf
[cite_start]

Scroll paling bawah, tambahkan[cite: 269]:

subnet 10.10.10.0 netmask 255.255.255.0 { range 10.10.10.10 10.10.10.100; option routers 10.10.10.1; option domain-name-servers 10.10.10.1, 8.8.8.8; }

3. Start Service

root@fw:~# systemctl restart isc-dhcp-server
root@fw:~# systemctl status isc-dhcp-server
