Modul Security

Keamanan Jaringan (IPTables)

Kita akan mengamankan Router dari serangan luar dengan memblokir Ping (ICMP) dan membatasi akses SSH hanya untuk Admin.

Skenario Keamanan

Rule 1: Block Ping (ICMP)

Router tidak boleh membalas ping dari Client, tapi Client tetap bisa internetan.

Client -> Ping Router (10.10.10.1) -> DROP

Rule 2: Secure SSH

SSH ke Router hanya diizinkan dari jaringan lokal (Management), akses dari luar ditolak.

Client -> SSH Router (Port 22) -> ACCEPT
Internet -> SSH Router -> DROP
1

Cek Status Awal

Pastikan tool firewall terinstall di Router (fw).

Di terminal Router (fw), pastikan paket penyimpan rule ada:

root@fw:~# apt install iptables-persistent -y

Cek aturan saat ini (Harusnya ada rule NAT dari Fase 2):

root@fw:~# iptables -t nat -L -v -n
2

Blokir Ping (ICMP)

Membuat router "Invisible" dari Client.

Penerapan Rule

Perintah ini akan membuang paket Ping (echo-request) yang masuk ke Router dari arah Client (ens39).

# Format: iptables -A INPUT -i [Interface] -p icmp -j DROP
root@fw:~# iptables -A INPUT -i ens39 -p icmp --icmp-type echo-request -j DROP

Tes Langsung (Validasi)

Buka CMD di Laptop Client, coba ping Gateway:

ping 10.10.10.1

Target: Request Timed Out (RTO).
Tapi coba ping ke Google (8.8.8.8), harusnya tetap Reply.

3

Amankan SSH (Port 22)

Hanya izinkan Laptop Client, blokir yang lain.

Strategi Whitelist

Kita izinkan dulu network Client, baru blokir sisanya.

# 1. Izinkan Network Client (10.10.10.0/24) masuk SSH
root@fw:~# iptables -A INPUT -p tcp --dport 22 -s 10.10.10.0/24 -j ACCEPT

# 2. Blokir SSH dari manapun selain di atas
root@fw:~# iptables -A INPUT -p tcp --dport 22 -j DROP
4

Simpan Permanen

Agar aturan tidak hilang saat restart.

root@fw:~# netfilter-persistent save

Jika Anda ingin menghapus semua aturan dan mulai dari nol: iptables -F lalu simpan lagi.

Verifikasi Akhir

1. Ping Router

Dari Laptop, ping ke 10.10.10.1.

Harus RTO
2. Ping Internet

Dari Laptop, ping ke 8.8.8.8.

Harus Reply
3. SSH Router

Buka PuTTY ke 10.10.10.1.

Bisa Login
