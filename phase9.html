Virtual Private Network (VPN)

Membangun terowongan aman (Tunneling) agar Client bisa mengakses jaringan Internal dan DMZ dari luar seolah-olah sedang berada di dalam kantor.

Target Konfigurasi

VPN Details:

  • Server: Router (fw)
  • IP VPN Server: 10.10.10.1 (IP Gateway)
  • Client IP Pool: 10.10.10.200 - 210
  • Username: lks
  • Password: vpn123

Keuntungan PPTP:

  • Instalasi sangat cepat.
  • Ringan untuk resource VM.
  • Native: Langsung didukung Windows tanpa install software (seperti OpenVPN).
1

Install Poptop

Software PPTP Server di Debian.

Lakukan di Router (fw). Pastikan DVD-1/2 siap.

root@fw:~# apt install pptpd -y
2

Konfigurasi IP Pool

Menentukan IP untuk "Terowongan".

root@fw:~# nano /etc/pptpd.conf

Cari baris paling bawah, tambahkan/aktifkan:

localip 10.10.10.1 remoteip 10.10.10.200-210
3

Membuat Akun VPN

User & Password untuk login.

root@fw:~# nano /etc/ppp/chap-secrets

Gunakan format: "user" [tab] pptpd [tab] "password" [tab] *

# client server secret IP addresses lks pptpd vpn123 *
root@fw:~# systemctl restart pptpd

Pengujian di Windows (Laptop)

Cara Setup di Windows 10/11:

  1. Cari VPN Settings di menu Start.
  2. Klik Add a VPN Connection.
  3. VPN Provider: Windows (built-in).
  4. Connection Name: LKS-VPN.
  5. Server Name/Address: 10.10.10.1 (IP Router).
  6. VPN Type: PPTP.
  7. Username: lks | Password: vpn123.
  8. Klik Save lalu klik Connect.

Verifikasi Sukses (CMD Laptop):

Jika sudah "Connected", cek IP VPN Anda:

C:\> ipconfig PPP adapter LKS-VPN: IPv4 Address. . . . . . . . . . . : 10.10.10.200

Troubleshooting VPN

Gagal Konek?

  • Firewall IPTables: Pastikan Anda tidak memblokir protokol GRE atau port 1723. Coba ketik iptables -F sementara untuk tes.
  • MS-DNS: Jika butuh internet saat VPN, edit /etc/ppp/pptpd-options tambahkan ms-dns 8.8.8.8.
  • Conflict: Pastikan Wi-Fi mati agar tidak rebutan jalur gateway.
