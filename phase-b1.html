Konfigurasi Switch (Layer 2)

Memecah switch fisik menjadi beberapa jaringan logis (VLAN) dan menghubungkan antar switch menggunakan jalur Trunk.

Pahami Konsepnya

VLAN (Access)

Seperti sekat ruangan. Port Access hanya mengizinkan 1 VLAN (contoh: PC Siswa hanya boleh masuk Ruang 10).

Trunking

Jalan tol antar gedung. Trunk membawa paket dari banyak VLAN sekaligus menggunakan label (Tagging 802.1Q).

Native VLAN

Trafik tanpa label di jalan tol. Defaultnya VLAN 1. Demi keamanan, kita pindahkan ke VLAN 99.

Recap Topologi Fisik (Fase B0):

  • SW-1 (Switch Kiri): Terhubung ke Router.
  • SW-2 (Switch Kanan): Hanya terhubung ke SW-1.
  • Fa0/2: Kabel Trunk (Jembatan antar switch).
  • Fa0/5 & 10: Ke PC Client.
1

Buat Database VLAN

Wajib dilakukan di KEDUA Switch (SW-1 dan SW-2).

Switch> enable Switch# configure terminal Switch(config)# hostname SW-1 ! (Ganti SW-2 saat config di switch kanan) ! 1. Membuat Kamar Siswa SW-1(config)# vlan 10 SW-1(config-vlan)# name SISWA SW-1(config-vlan)# exit ! 2. Membuat Kamar Guru SW-1(config)# vlan 20 SW-1(config-vlan)# name GURU SW-1(config-vlan)# exit ! 3. Membuat Native VLAN (Keamanan) SW-1(config)# vlan 99 SW-1(config-vlan)# name NATIVE SW-1(config-vlan)# exit
2

Assign Port Access (PC)

Memasukkan PC ke kamarnya. Lakukan di SW-1 dan SW-2.

Lihat Topologi: PC Kiri & PC Kanan.

! Port Fa0/5 untuk Siswa (VLAN 10) SW-1(config)# interface fa0/5 SW-1(config-if)# switchport mode access SW-1(config-if)# switchport access vlan 10 SW-1(config-if)# exit ! Port Fa0/10 untuk Guru (VLAN 20) SW-1(config)# interface fa0/10 SW-1(config-if)# switchport mode access SW-1(config-if)# switchport access vlan 20 SW-1(config-if)# exit
3

Konfigurasi Trunking (Jalan Tol)

Jalur antar perangkat jaringan. Konfigurasi berbeda tiap switch!

A. Konfigurasi SW-1 (Switch Kiri)

SW-1 punya 2 kaki Trunk: Fa0/1 (ke Router) & Fa0/2 (ke SW-2).

SW-1(config)# interface range fa0/1-2 SW-1(config-if-range)# switchport mode trunk SW-1(config-if-range)# switchport trunk native vlan 99

B. Konfigurasi SW-2 (Switch Kanan)

SW-2 hanya punya 1 kaki Trunk: Fa0/2 (ke SW-1).

SW-2(config)# interface fa0/2 SW-2(config-if)# switchport mode trunk SW-2(config-if)# switchport trunk native vlan 99
WARNING: Jika muncul pesan "Native VLAN mismatch", itu wajar. Itu artinya satu switch sudah disetting VLAN 99, satunya masih VLAN 1. Pesan akan hilang setelah kedua switch selesai disetting.

Verifikasi Hasil

1. Cek Tabel VLAN
SW-1# show vlan brief

Pastikan:
- VLAN 10 (SISWA) aktif, anggota: Fa0/5
- VLAN 20 (GURU) aktif, anggota: Fa0/10

2. Cek Interface Trunk
SW-1# show interfaces trunk

Pastikan:
- Port Fa0/1 dan Fa0/2 muncul di daftar.
- Status: trunking
- Native VLAN: 99

Kembali ke Cabling Lanjut ke Fase B2 (Inter-VLAN Routing)
