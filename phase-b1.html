Modul Cisco

Dasar Switching (L2)

Membangun pondasi jaringan dengan memisahkan trafik (VLAN) dan menghubungkan antar-switch (Trunking) secara aman.

Pahami Konsepnya

Apa itu VLAN?

Bayangkan Sekolah. VLAN itu seperti "Ruang Kelas". Siswa kelas 10 tidak boleh mendengar percakapan kelas 11. Secara fisik mereka di gedung yang sama (Switch), tapi secara logik mereka terpisah.

Apa itu Trunk?

Trunk itu seperti "Jalan Tol" antar gedung. Jika siswa kelas 10 ingin pindah ke gedung sebelah, mereka harus lewat jalan tol ini. Di jalan tol, setiap siswa diberi "Stempel/Tag" (802.1Q) agar satpam gedung sebelah tahu mereka dari kelas mana.

Apa itu Native VLAN?

Di jalan tol (Trunk), Native VLAN adalah satu-satunya trafik yang tidak diberi stempel (Untagged). Defaultnya VLAN 1. Untuk keamanan (mencegah VLAN Hopping Attack), kita pindahkan ke VLAN yang tidak terpakai (misal VLAN 99).

Tabel Perencanaan

VLAN ID Nama Port Anggota (Access) Fungsi
10 SISWA Fa0/1 - Fa0/10 Jaringan Siswa
20 GURU Fa0/11 - Fa0/20 Jaringan Guru
99 NATIVE - Keamanan Trunk (Native)

Topologi: Kabel Cross menghubungkan SW-1 (Gi0/1) ke SW-2 (Gi0/1).

1

Setup Database VLAN

Lakukan konfigurasi ini di SEMUA SWITCH (SW-1 dan SW-2).

Switch> enable Switch# configure terminal Switch(config)# hostname SW-1 ! Ganti nama agar tidak bingung ! 1. Membuat Kamar Siswa SW-1(config)# vlan 10 SW-1(config-vlan)# name SISWA SW-1(config-vlan)# exit ! 2. Membuat Kamar Guru SW-1(config)# vlan 20 SW-1(config-vlan)# name GURU SW-1(config-vlan)# exit ! 3. Membuat Native VLAN (Wajib ada meski tidak ada PC-nya) SW-1(config)# vlan 99 SW-1(config-vlan)# name NATIVE SW-1(config-vlan)# exit
2

Assign Port Access

Mendaftarkan port kabel ke VLAN yang sesuai. Ingat: Port Access hanya untuk PC/Laptop.

! Mendaftarkan Port 1 sampai 10 ke Siswa SW-1(config)# interface range fa0/1-10 SW-1(config-if-range)# switchport mode access SW-1(config-if-range)# switchport access vlan 10 SW-1(config-if-range)# exit ! Mendaftarkan Port 11 sampai 20 ke Guru SW-1(config)# interface range fa0/11-20 SW-1(config-if-range)# switchport mode access SW-1(config-if-range)# switchport access vlan 20 SW-1(config-if-range)# exit
3

Konfigurasi Trunk (Jalan Tol)

Konfigurasi paling krusial. Salah setting di sini, antar switch tidak terkoneksi.

PENTING: Konfigurasi Native VLAN harus SAMA PERSIS di kedua switch. Jika SW-1 Native 99 tapi SW-2 Native 1, akan terjadi error "Native VLAN Mismatch".
! Masuk ke port penghubung antar switch (misal Gigabit 0/1) SW-1(config)# interface gi0/1 ! 1. Aktifkan Mode Trunk SW-1(config-if)# switchport mode trunk ! 2. Security: Ubah Native VLAN ke 99 SW-1(config-if)# switchport trunk native vlan 99 ! 3. Security: Filter VLAN yang Boleh Lewat (Whitelist) SW-1(config-if)# switchport trunk allowed vlan 10,20,99

ulangi konfigurasi trunk ini pada switch lawan (SW-2) di port pasangannya.

Verifikasi Hasil

1. Cek Tabel VLAN
SW-1# show vlan brief

Pastikan VLAN 10, 20, 99 Aktif. Pastikan port Fa0/1-10 ada di baris VLAN 10.

2. Cek Status Trunk
SW-1# show interfaces trunk

Port: Gi0/1 harus muncul.
Mode: on
Encapsulation: 802.1q
Native VLAN: 99 (Pastikan bukan 1)
