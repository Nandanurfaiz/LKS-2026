Keamanan & Translasi Jaringan

Tahap akhir: Konfigurasi keamanan di R1 untuk membagi zona **Inside** (LAN) dan **Outside** (Internet).

1

Konfigurasi NAT Overload

Device: R1 (Gateway)

A. Penentuan Interface (Inside vs Outside)

R1(config)# interface range fa0/0.10, fa0/0.20 R1(config-if-range)# ip nat inside R1(config-if-range)# exit R1(config)# interface loopback 0 R1(config-if)# ip nat outside

B. Aturan Izin NAT

R1(config)# access-list 1 permit 192.168.10.0 0.0.0.255 R1(config)# access-list 1 permit 192.168.20.0 0.0.0.255 R1(config)# ip nat inside source list 1 interface loopback 0 overload
2

Enkripsi Remote (SSH)

Supaya User Guru bisa setting router tanpa kabel console.

R1(config)# ip domain-name lks2026.id R1(config)# crypto key generate rsa ! (Pilih 1024) R1(config)# username admin privilege 15 secret lks123 R1(config)# line vty 0 4 R1(config-line)# transport input ssh
3

ACL Barrier (Pembatasan)

Melarang Siswa melakukan Ping ke Router.

R1(config)# access-list 100 deny icmp 192.168.10.0 0.0.0.255 host 192.168.10.1 R1(config)# access-list 100 permit ip any any R1(config)# interface fa0/0.10 R1(config-subif)# ip access-group 100 in

Uji Coba Terakhir

Tes 1

Ping 8.8.8.8 dari PC Siswa. Harus Reply.

Tes 2

Ping 192.168.10.1 dari Siswa. Harus Fail (ACL).

Tes 3

SSH ke 192.168.20.1 dari PC Guru. Harus Success.
