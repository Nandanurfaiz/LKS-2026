Keamanan & Koneksi WAN

Menghubungkan jaringan lokal (Private IP) ke Internet (Public IP) secara aman menggunakan NAT dan membatasi akses dengan Access Control Lists (ACL).

NAT (Network Address Translation)

Bayangkan kantor besar. Karyawan punya nomor ekstensi telepon masing-masing (Private IP), tapi saat menelepon keluar, penerima hanya melihat Satu Nomor Kantor Pusat (Public IP).

PAT (Overload) = Banyak PC, 1 IP Public.

ACL (Access Control List)

Seperti Satpam di pintu masuk. Satpam membawa daftar aturan: "Boleh masuk pakai kemeja (HTTP), tapi DILARANG pakai kaos oblong (Ping/ICMP)".

Extended ACL = Satpam Galak & Teliti.
1

Konfigurasi NAT Overload (PAT)

Mengizinkan PC Siswa & Guru mengakses Internet.

R1(config)# access-list 1 permit 192.168.10.0 0.0.0.255 R1(config)# access-list 1 permit 192.168.20.0 0.0.0.255 ! 1. Tentukan Interface Dalam (INSIDE) R1(config)# interface g0/0 R1(config-if)# ip nat inside R1(config-if)# exit ! 2. Tentukan Interface Luar (OUTSIDE) R1(config)# interface serial0/0/0 R1(config-if)# ip nat outside R1(config-if)# exit ! 3. Aktifkan Translation Rule (MANTRA UTAMA) R1(config)# ip nat inside source list 1 interface serial0/0/0 overload
Troubleshooting: Jika internet tidak jalan, cek apakah `ip nat inside` sudah dipasang di interface G0/0 (induk) atau sub-interface. Biasanya cukup di induknya.
2

Extended ACL (Firewall)

Skenario LKS: Block Ping ke Server, tapi Boleh Buka Web Server.

Asumsi: Server IP adalah 202.10.10.10

! Membuat Aturan Nomor 100 (Extended) R1(config)# access-list 100 deny icmp any host 202.10.10.10 R1(config)# access-list 100 permit tcp any host 202.10.10.10 eq www R1(config)# access-list 100 permit ip any any ! Menerapkan Aturan ke Pintu Masuk (Inbound) R1(config)# interface g0/0 R1(config-if)# ip access-group 100 in
3

NTP Client

Sinkronisasi waktu router dengan server internet.

! Mengatur Zona Waktu (WIB = GMT+7) R1(config)# clock timezone WIB 7 ! Menunjuk Server NTP R1(config)# ntp server 202.10.10.10 ! Verifikasi (Tunggu beberapa menit) R1# show ntp status R1# show clock

Verifikasi Akhir Modul B

1. Cek Tabel NAT

Buka Browser di PC Siswa, akses google.com atau IP Server. Lalu cek router:

R1# show ip nat translations

Harus muncul daftar terjemahan dari Inside Global (IP Public) ke Inside Local (IP Private 192.168.x.x).

2. Pengujian ACL

  • Ping Server: Dari PC, Ping ke 202.10.10.10 ➔ Hasil: Destination Host Unreachable (Diblokir Router).
  • Web Server: Dari PC, Buka Browser ke 202.10.10.10 ➔ Hasil: Halaman Web Muncul (Diizinkan).
