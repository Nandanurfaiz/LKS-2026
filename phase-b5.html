Modul B

Menghubungkan ke Dunia Luar

Agar PC Client bisa internetan, kita butuh NAT (Network Address Translation). Kita juga akan memasang ACL sebagai satpam jaringan dan NTP agar jam di semua perangkat akurat.

1

Konfigurasi NAT Overload (PAT)

Mengubah IP Lokal (192.168.x.x) menjadi IP Publik saat ke Internet.

Buka Router HQ. Kita perlu menentukan mana pintu "Masuk" dan mana pintu "Keluar".

! 1. Tentukan Interface Inside (Lokal)
HQ(config)# int g0/1.10
HQ(config-subif)# ip nat inside
HQ(config-subif)# exit
HQ(config)# int g0/1.20
HQ(config-subif)# ip nat inside
HQ(config-subif)# exit

! 2. Tentukan Interface Outside (Internet)
HQ(config)# int g0/0
HQ(config-if)# ip nat outside
HQ(config-if)# exit

! 3. Buat Izin (Access List) siapa yang boleh di-NAT
HQ(config)# access-list 1 permit 192.168.10.0 0.0.0.255
HQ(config)# access-list 1 permit 192.168.20.0 0.0.0.255

! 4. Aktifkan NAT Overload (PAT)
HQ(config)# ip nat inside source list 1 interface g0/0 overload
2

Sinkronisasi Waktu (NTP)

Agar log keamanan valid, jam harus akurat.

A. Router ISP (Master)

! ISP jadi sumber waktu (Stratum 1)
ISP(config)# ntp master 1

B. Router HQ (Client)

! Minta jam ke ISP
HQ(config)# ntp server 10.10.10.2
HQ(config)# do show ntp status
3

Keamanan Jaringan (ACL)

Memblokir akses yang tidak diinginkan.

Skenario: PC Siswa (VLAN 10) DILARANG ping ke Google DNS (8.8.8.8), tapi BOLEH buka web (HTTP).

HQ(config)# access-list 100 deny icmp 192.168.10.0 0.0.0.255 host 8.8.8.8
HQ(config)# access-list 100 permit ip any any

! Pasang satpam di pintu masuk VLAN 10
HQ(config)# int g0/1.10
HQ(config-subif)# ip access-group 100 in
Ingat: Selalu akhiri ACL dengan permit ip any any, jika tidak, semua trafik lain akan terblokir otomatis (Implicit Deny).
4

Verifikasi

Cek hasil konfigurasi.

  • Tes NAT: Dari PC Siswa, ping 8.8.8.8. Di HQ ketik show ip nat translations. Harusnya muncul daftar IP Publik.
  • Tes ACL: Dari PC Siswa, ping 8.8.8.8 harus Request Timed Out / Destination Unreachable. Tapi browsing ke http://8.8.8.8 (jika ada web) harus bisa.
  • Tes NTP: Ketik show clock di HQ, pastikan jam sudah sama dengan ISP.
