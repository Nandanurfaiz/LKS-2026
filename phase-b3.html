Redundansi Jaringan (L2)

Memastikan jaringan tetap hidup meskipun salah satu kabel putus, serta mencegah terjadinya looping data.

EtherChannel (LACP)

Bayangkan jalan raya 1 lajur. Jika macet, semua berhenti. EtherChannel menggabungkan 2 kabel fisik menjadi 1 Jalan Tol Lebar.

  • Speed: 1 Gbps + 1 Gbps = 2 Gbps.
  • Redundancy: 1 kabel putus, koneksi tetap jalan.

Spanning Tree (STP)

Jika ada 2 kabel antar switch tanpa EtherChannel, data akan berputar-putar (Loop) sampai switch hang. STP bertugas sebagai Polisi Lalu Lintas yang menutup salah satu jalan untuk mencegah Loop.

Rapid-PVST = Polisi Modern (Cepat Tanggap)
1

EtherChannel (LACP)

Menggabungkan Port Gi0/1 dan Gi0/2 menjadi Port-Channel 1.

Lakukan di KEDUA Switch (SW-1 dan SW-2):

SW-1(config)# interface range gi0/1-2 SW-1(config-if-range)# channel-group 1 mode active SW-1(config-if-range)# no shutdown SW-1(config-if-range)# exit ! Konfigurasi Logical Interface (Port-Channel 1) SW-1(config)# interface port-channel 1 SW-1(config-if)# switchport mode trunk SW-1(config-if)# switchport trunk native vlan 99 SW-1(config-if)# switchport trunk allowed vlan 10,20,99
PENTING: Konfigurasi Trunk (Native/Allowed VLAN) harus dilakukan di dalam interface port-channel 1, bukan lagi di fisik Gi0/1 atau Gi0/2.
2

Rapid-PVST Configuration

Mengubah mode STP menjadi cepat dan menentukan Root Bridge.

Di SW-1 (Switch Utama):

SW-1(config)# spanning-tree mode rapid-pvst SW-1(config)# spanning-tree vlan 10,20,99 root primary

Di SW-2 (Switch Cadangan/Cabang):

SW-2(config)# spanning-tree mode rapid-pvst SW-2(config)# spanning-tree vlan 10,20,99 root secondary
3

Port Security (Hardening)

Mencegah orang asing colok kabel sembarangan (Optional tapi Poin Tinggi).

Terapkan pada port Access (misal fa0/1):

SW-1(config)# interface range fa0/1-10 SW-1(config-if-range)# switchport port-security SW-1(config-if-range)# switchport port-security maximum 1 SW-1(config-if-range)# switchport port-security mac-address sticky SW-1(config-if-range)# switchport port-security violation restrict

Verifikasi

1. Cek EtherChannel

SW-1# show etherchannel summary

Cari kode SU di samping Po1.
S = Layer 2
U = In Use (Aktif)

2. Cek Spanning Tree

SW-1# show spanning-tree

Pastikan tertulis: "This bridge is the root" pada SW-1.
