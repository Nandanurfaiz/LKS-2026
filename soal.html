Lomba Kompetensi Siswa (LKS) 2026

IT Network System Administration

A. Skenario Proyek

"Perusahaan **Kebumen Digital Solution** sedang melakukan migrasi infrastruktur. Sebagai Network Engineer, Anda bertugas membangun server Linux yang redundan (High Availability) dan mengamankan jaringan Cisco menggunakan VLAN, OSPF, dan Firewall. Seluruh layanan harus dapat diakses melalui domain internal lks2026.id dengan aman dan stabil."

B. Algoritma Pengerjaan (Workflow)

I. Linux Workflow

1
Inisiasi System

Config IP Static, Hostname, & Repo Lokal pada semua Node.

2
Router & Gateway

IP Forwarding & NAT Masquerade pada VM Router.

3
DNS & Web

Bind9 (lks2026.id) & Nginx Web Server (Node 1 & 2).

4
HAProxy LB

Stop Nginx Router! Config HAProxy RoundRobin.

5
Security & LDAP

LDAP, Samba, Firewall, & VPN Server.

II. Cisco Workflow

1
L2 Switching

VLAN, Trunking (Native 99), & EtherChannel.

2
L3 Routing

Inter-VLAN (RoaS) & DHCP Server.

3
OSPF Area 0

Advertise Network, Passive Interface, Default Route.

4
WAN Security

NAT Overload (PAT), ACL, & Port Security.

A1. Web & Infrastructure

  • Install Bind9 & buat Zone lks2026.id.
  • Install Nginx pada web-node1 & web-node2 dengan konten index.html yang berbeda (identitas node).
  • Konfigurasi HAProxy (Port 80) dengan algoritma Round Robin ke backend web.
  • Pastikan Failover berjalan (Web tetap akses meski 1 node mati).

A2. Security & Storage

  • Deploy OpenLDAP Server & PHPLDAPAdmin.
  • Konfigurasi Samba Share "Public" & "Secure" (Auth via LDAP).
  • Iptables: Drop ICMP (Ping) dari Client ke Router. Allow SSH from Internal.
  • Setup VPN Server (PPTP/Wireguard) network 10.10.10.0/24.
