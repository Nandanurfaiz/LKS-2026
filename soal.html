BAGIAN 1: MODUL A (Linux Environment)

A. Skenario Modul A

Anda bertindak sebagai Administrator Jaringan di perusahaan "Kebumen Tech". Anda diminta membangun infrastruktur jaringan yang aman dengan memisahkan layanan publik (DMZ) dan layanan internal (INT). Anda harus mengkonfigurasi Router, DNS, Web Server dengan Load Balancing, Mail Server, serta layanan direktori (LDAP) dan File Sharing (Samba). Seluruh sistem harus berjalan di atas OS Debian 12 menggunakan VMWare Workstation.

B. Konfigurasi IP & Domain (Modul A)

Router (fw) DMZ 172.16.10.1/24 -

Web01 & Web02 DMZ 172.16.10.10 & .20 172.16.10.1

C. Job Sheet Modul A

1. Router & Firewall

  • IP Forwarding & NAT (Masquerade).
  • DHCP Server (Interface Client).
  • Block PING Client ke DMZ (Allow Web).
  • SSH ke Router hanya dari INT.

2. DNS, Web & LB

  • DNS Zone: lks2026.id.
  • Web01 & Web02 (Nginx/Apache).
  • HAProxy Load Balancer (Round Robin).
  • HTTPS Self-signed Certificate.

3. Internal Services

  • Mail Server (Postfix/Dovecot).
  • Webmail (Roundcube).
  • LDAP Server (User Management).
  • Samba (Integrasi LDAP).

4. Remote Access

  • VPN Server (Wireguard/OpenVPN).
  • Client Connect to VPN.

BAGIAN 2: MODUL B (Cisco Packet Tracer)

A. Skenario Modul B

"Bank Jaringan Kebumen" membutuhkan infrastruktur jaringan baru yang redundant dan aman. Anda diminta merancang jaringan menggunakan Cisco Packet Tracer. Jaringan terdiri dari kantor pusat (HQ) yang memiliki 3 VLAN (Management, Staff, Server) dan terhubung ke Router ISP. Anda harus menerapkan Dynamic Routing OSPF, Redundansi Gateway (FHRP), serta keamanan Port Security pada Switch akses.

B. Tabel VLAN & Network (Modul B)

VLAN ID Nama VLAN Network ID Gateway (HSRP)
10 Management 10.10.10.0/24 10.10.10.254
20 Staff_Bank 10.10.20.0/24 10.10.20.254
30 Server_Farm 10.10.30.0/24 10.10.30.254
99 Native/Parking - -

C. Job Sheet Modul B

1

Switching Technologies

Implementasi L2 Security & Management pada Switch Distribusi (DSW) dan Akses (ASW).

  • Konfigurasi VLAN Database & VTP Mode Client/Server (Domain: lks2026).
  • Konfigurasi Trunking (802.1q) & Native VLAN 99.
  • Konfigurasi EtherChannel (LACP) antar Switch Distribusi.
  • Port Security: Max 2 MAC Address, Violation Shutdown pada port akses.
2

Routing & Redundancy (L3)

Implementasi Routing Dinamis dan Gateway Redundancy pada Router/L3 Switch.

  • Konfigurasi Inter-VLAN Routing.
  • Konfigurasi OSPF Process ID 10 Area 0. Pastikan semua network ter-advertise.
  • Konfigurasi FHRP (HSRP) pada VLAN 10, 20, 30. Router 1 menjadi Active Gateway untuk VLAN 10, Router 2 untuk VLAN 20 (Load Balancing).
3

Services & WAN

  • Konfigurasi DHCP Server pada Router HQ untuk VLAN Staff & Management (Exclude 10 IP pertama).
  • Konfigurasi NAT Overload (PAT) agar VLAN Staff bisa ping ke Internet (8.8.8.8).
  • Konfigurasi SSH Version 2 pada semua perangkat jaringan (User: admin, Pass: lks2026).
  • Konfigurasi NTP Server (arahkan ke server internet).
