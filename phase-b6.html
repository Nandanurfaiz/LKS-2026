Modul B

Pengamanan Perangkat (Hardening)

Tahap terakhir: Mengganti Telnet yang usang dengan SSH (Encrypted), memblokir perangkat asing di Switch menggunakan Port Security, dan mengaktifkan monitoring SNMP.

1

Secure SSH Access

Wajib digunakan untuk manajemen router jarak jauh.

Buka Router HQ. Kita harus membuat kunci enkripsi (Crypto Key) terlebih dahulu.

! 1. Syarat SSH: Harus ada Hostname & Domain
HQ(config)# hostname HQ
HQ(config)# ip domain-name lks2026.id

! 2. Buat Kunci RSA (Min 1024 bit)
HQ(config)# crypto key generate rsa
Input bits: 1024

! 3. Buat User Admin
HQ(config)# username admin privilege 15 secret cisco123

! 4. Paksa Login via SSH (Matikan Telnet)
HQ(config)# line vty 0 4
HQ(config-line)# login local
HQ(config-line)# transport input ssh
Tes: Dari PC Admin, buka Command Prompt -> ssh -l admin 192.168.10.1. Jika diminta password, berarti sukses!
2

Port Security (Sticky MAC)

Mencegah orang asing colok kabel sembarangan.

Buka Switch 1. Kita akan amankan port yang mengarah ke PC Siswa (misal F0/10).

SW1(config)# int f0/10
! Port harus mode access, tidak boleh trunk/dynamic
SW1(config-if)# switchport mode access
SW1(config-if)# switchport port-security
! Maksimal 1 perangkat saja
SW1(config-if)# switchport port-security maximum 1
! Ingat MAC Address perangkat pertama yang colok
SW1(config-if)# switchport port-security mac-address sticky
! Hukuman jika melanggar: Matikan Port
SW1(config-if)# switchport port-security violation shutdown
Skenario Normal
PC Siswa (Resmi) terhubung -> Port UP (Lampu Hijau). Switch mencatat MAC Address PC Siswa.
Skenario Serangan
Kabel dicabut, dicolok ke Laptop Hacker -> Port otomatis mati (Shutdown). Lampu Merah.
3

SNMP & Banner

Monitoring dan Legal Notice.

A. Konfigurasi SNMP

Agar router bisa dipantau oleh server monitoring (Cacti/PRTG).

HQ(config)# snmp-server community lks-ro ro
! 'lks-ro' adalah password, 'ro' artinya Read Only (Hanya baca data)

B. Banner & Password Encryption

Pesan peringatan hukum & menyandikan semua password.

HQ(config)# banner motd # DILARANG MASUK SELAIN ADMIN! #
HQ(config)# service password-encryption
