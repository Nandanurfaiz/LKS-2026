Instalasi "Golden Master"

Panduan detail instalasi Debian 12 satu kali untuk dikloning menjadi 5 Server. Menggunakan parameter Peserta 01 sebagai simulasi.

Mengapa "Peserta 01"?

[cite_start]Di naskah soal[cite: 38], username dan password menggunakan variabel XX (contoh: Network-XX).
Saat lomba, Anda akan mendapat nomor undian (misal 05, 10, atau 12).
Untuk latihan ini, kita sepakati Anda adalah Peserta No. 01. Jadi semua XX diganti angka 01.

1

Setup Jaringan (Virtual Network Editor)

Buka Virtual Network Editor di VMware (Run as Admin). Pastikan konfigurasi persis seperti ini agar sesuai dengan topologi fisik Fase 1:

VMnet0 Bridged

Hubungkan ke Physical LAN/WiFi laptop Anda yang ada internetnya.

VMnet10 Host-Only

Jalur DMZ.
Subnet: Sesuaikan IP Soal.
Matikan DHCP VMware!

VMnet11 Host-Only

Jalur INT.
Subnet: Sesuaikan IP Soal.
Matikan DHCP VMware!

Proses Instalasi Debian

A. Booting & Bahasa

  • Pilih Install (Jangan Graphical Install agar ringan).
  • Language: English.
  • Location: United States (atau Other -> Asia -> Indonesia).
  • Keymap: American English.

B. Network & Hostname

  • Primary Interface: Pilih yang ens33 (atau yang paling atas/bridged).
  • Hostname: master-debian (Nanti akan kita ubah setelah clone).
  • Domain name: lks2026.id (Sesuai soal Fase 3).

C. Users & Passwords (CRITICAL)

Full Name: peserta01

Username: peserta01

Password: Juara-2026

D. Partitioning (Manual)

Wajib Manual
  1. Pilih Manual.
  2. Pilih Harddisk (SCSI3/sda) -> Yes (Create new empty partition table).
  3. Buat SWAP:
    • Pilih Free Space -> Create a new partition.
    • Size: 4 GB -> Primary -> Beginning.
    • Use as: swap area.
    • Done setting up the partition.
  4. Buat ROOT (/):
    • Pilih Free Space (Sisa) -> Create a new partition.
    • Size: (Biarkan default/max) -> Primary.
    • Use as: Ext4 journaling file system.
    • Mount point: /.
    • Done setting up the partition.
  5. Pilih Finish partitioning and write changes to disk -> Yes.

E. Software Selection

Scan extra media? No. Use network mirror? No.

Pilih paket dengan tombol SPASI:

  • [ ] Debian desktop environment
  • [ ] ... GNOME ...
  • [*] SSH server
  • [*] Standard system utilities
4

Finalisasi Master & Cloning

Setelah instalasi selesai dan reboot, login sebagai root. Lakukan hal ini sebelum Cloning:

1. Daftarkan DVD Repo (Penting!)

Pastikan ISO Debian terhubung (Connected) di setting VMware.

apt-cdrom add # Tekan Enter. Ini akan membaca paket dari ISO. apt update

2. Install Tools Dasar

apt install vim nano net-tools dnsutils iputils-ping -y

3. Matikan & Clone

poweroff

Di VMware: Klik Kanan VM Master -> Manage -> Clone -> Full Clone.
Buat 5 Clone dengan nama: FW, INT-SRV, MAIL, WEB01, WEB02.
