Layer 3

Open Shortest Path First (OSPF)

Mengaktifkan protokol routing dinamis agar Router R1 bisa "mengumumkan" keberadaan jaringan Siswa dan Guru ke dunia luar (Internet/WAN).

Pahami Konsepnya

Area 0 (Backbone)

OSPF membagi jaringan menjadi area-area. Area 0 adalah pusat/inti. Semua router harus terhubung ke Area 0.

Wildcard Mask

Kebalikan dari Subnet Mask. Digunakan OSPF untuk mencocokkan IP.
Subnet: 255.255.255.0 â®• Wildcard: 0.0.0.255.

Persiapan WAN (Simulasi Internet)

Karena di topologi ini kita hanya punya 1 Router, kita perlu membuat Interface Loopback untuk berpura-pura menjadi koneksi Internet (ISP).

R1(config)# interface loopback 0 R1(config-if)# ip address 8.8.8.8 255.255.255.255 R1(config-if)# exit
1

Aktifkan OSPF Process

Mulai layanan OSPF pada Router R1.

R1(config)# router ospf 1 R1(config-router)# router-id 1.1.1.1 ! Advertise Network (Umumkan jaringan kita ke tetangga) R1(config-router)# network 192.168.10.0 0.0.0.255 area 0 R1(config-router)# network 192.168.20.0 0.0.0.255 area 0 R1(config-router)# network 192.168.99.0 0.0.0.255 area 0 ! Advertise Internet (Loopback) R1(config-router)# network 8.8.8.8 0.0.0.0 area 0
2

Security: Passive Interface

Mencegah router "mengobrol" OSPF ke arah PC Client (Bahaya & Boros Bandwidth).

Kita ingin OSPF hanya jalan ke arah Router ISP, jangan ke arah LAN Siswa/Guru.

R1(config-router)# passive-interface fa0/0.10 R1(config-router)# passive-interface fa0/0.20 R1(config-router)# passive-interface fa0/0.99
Tips Pro: Cara lebih cepat adalah passive-interface default (matikan semua), lalu no passive-interface [Interface_ke_ISP].
3

Sebarkan Default Route

Memberitahu router lain: "Kalau bingung mau kemana, lewat saya saja (karena saya punya internet)."

R1(config-router)# default-information originate R1(config-router)# exit

Verifikasi Hasil

1. Cek Protokol
R1# show ip protocols

Pastikan:
- Routing Protocol is "ospf 1"
- Router ID: 1.1.1.1
- Routing for Networks: (Daftar IP VLAN 10, 20, 99 muncul).

2. Cek Interface
R1# show ip ospf interface brief

Pastikan interface yang aktif OSPF muncul di sini. Interface Loopback 0 harusnya statusnya LOOP/DR.
