| Setup Infrastruktur & Master VM
Bobot 70% (Base)

Membangun Fondasi Lab

Fase ini menentukan keberhasilan fase berikutnya. Kita akan membangun "Jalur Kabel Virtual" sesuai soal LKS 2026 dan menginstall Master VM.

Target Soal (LKS 2026)

Identitas Jaringan:

Domain: lks2026.id
IP DMZ (Web): 172.16.10.x
IP INT (Server): 192.168.10.x
IP Client (Laptop): 10.10.10.x

Logika Topologi:

  • DMZ: Zona Publik untuk Web Server & Mail Server.
  • INT: Zona Internal untuk File Server (Samba).
  • Bridged (VMnet0): Wajib ada agar Laptop Fisik bisa terhubung ke Router VM.
1

Setting Virtual Network Editor

Langkah Kunci untuk Strategi B (Client Fisik).

1. Buka Aplikasi dengan Akses Admin

Cari "Virtual Network Editor" di menu Start Windows, klik kanan -> Run as Administrator.

2. Aktifkan Mode Edit

Klik tombol Change Settings di pojok kanan bawah jendela agar bisa mengedit.

3. Atur 3 Jalur Utama (PENTING!)

Pastikan konfigurasi VMnet persis seperti tabel di bawah ini:

VMnet Mode Subnet IP DHCP Fungsi
VMnet0 Bridged - - Client Fisik
VMnet10 Host-only 172.16.10.0 UNCHECK DMZ (Web/Mail)
VMnet11 Host-only 192.168.10.0 UNCHECK INT (Server)

4. Simpan

Klik Apply, tunggu restart network, lalu OK.

2

Instalasi Master VM (Offline)

Membuat induk untuk semua server.

2.1 Setup Awal VM

  • ISO Image: `debian-12.x.x-amd64-DVD-1.iso`
  • VM Name: `Debian-Master`
  • Disk Size: `20 GB` (Store virtual disk as a single file)
  • Memory: `1024 MB` (1 GB cukup untuk master CLI)
  • Network Adapter: Pilih `NAT` dulu (hanya saat install, nanti diubah).

2.2 Booting & Bahasa

Pilih Install (Mode Teks), Bahasa English, Lokasi Indonesia.

2.3 Identitas (Sesuai Soal)

Hostname master
Domain Name lks2026.id
  • Root Password: 123 (Biar cepat saat lomba)
  • Full Name/User: siswa
  • User Password: 123

2.4 Partisi Disk

Guided - use entire disk -> All files in one partition -> Finish -> Yes.

2.5 Package Manager (Mode DVD)

  • Scan extra installation media? -> YES
  • Use a network mirror? -> NO (Wajib Offline).
  • Participate in package survey? -> NO.

2.6 SOFTWARE SELECTION (WAJIB TELITI)

Gunakan tombol SPASI untuk uncheck/check. Pastikan HANYA 2 yang dicentang:

[ ] Debian desktop environment
[ ] ... GNOME
[ ] Web server
[*] SSH server   <-- YES
[*] Standard system utilities   <-- YES

2.7 Bootloader & Finish

Install GRUB -> Yes -> Pilih /dev/sda.
Reboot.

3

Konfigurasi Pasca-Install

Menyiapkan Repositori & Tools.

3.1 Login & Root

master login: siswa
Password: 123
siswa@master:~$ su -
Password: 123

3.2 Cek Repositori DVD

root@master:~# nano /etc/apt/sources.list
# Matikan semua link online (beri pagar #)
root@master:~# apt update

3.3 Install Paket Wajib (Tools)

Pastikan DVD-1 dimasukkan di VM Settings sebelum perintah ini.

root@master:~# apt install nano vim net-tools dnsutils iputils-ping curl wget zip unzip -y

3.4 Aktifkan SSH Root Login

Agar nanti bisa diremote pakai PuTTY/CMD.

root@master:~# nano /etc/ssh/sshd_config
Cari & Ubah:
#PermitRootLogin prohibit-password
Menjadi:
PermitRootLogin yes
root@master:~# systemctl restart ssh

3.5 Finalisasi

Matikan VM. Master siap dikloning.

root@master:~# apt clean
root@master:~# poweroff

Checklist Fase 1

VMnet0 diset Bridged (Ke Laptop Fisik).
VMnet10 (DMZ) & VMnet11 (INT) terbuat.
Master VM Domain: lks2026.id.
Repo DVD OK & SSH Root OK.
