Offline Strategy

Layanan Surat Elektronik

Fokus utama: Membuktikan Server bisa mengirim (SMTP) dan menerima (IMAP/POP3) email dari Laptop Client, meskipun tanpa Webmail.

Target Utama

VM Mail:

  • IP: 172.16.10.30 (VMnet10)
  • Gateway: 172.16.10.1
  • Domain: mail.lks2026.id

Metode Pengujian:

  • Utama: Telnet via CMD Laptop.
  • Alternatif: Thunderbird / Outlook.
  • Bonus: Roundcube (Jika paket tersedia).
1

Setup VM Mail

Konfigurasi Dasar.

1. IP Address (Pastikan Benar!)

root@mail:~# nano /etc/network/interfaces
auto ens33 iface ens33 inet static address 172.16.10.30/24 gateway 172.16.10.1 dns-nameservers 172.16.10.1
root@mail:~# hostnamectl set-hostname mail
root@mail:~# systemctl restart networking
2

Update DNS (Router)

Pindah ke VM fw.

1. Tambah Record Mail

root@fw:~# nano /etc/bind/db.lks

Gunakan format pendek (Cara 1) yang aman:

@ IN MX 10 mail mail IN A 172.16.10.30

2. Restart Bind9

root@fw:~# systemctl restart bind9
3

Install Postfix & Dovecot

Kembali ke VM mail.

1. Install Paket Inti

root@mail:~# apt install postfix dovecot-imapd dovecot-pop3d -y

2. Config Postfix (Krusial)

root@mail:~# nano /etc/postfix/main.cf

Tambahkan/Edit baris di bagian paling bawah:

home_mailbox = Maildir/ myhostname = mail.lks2026.id mynetworks = 127.0.0.0/8 [::ffff:127.0.0.0]/104 [::1]/128 172.16.10.0/24 10.10.10.0/24
  • home_mailbox: Format penyimpanan folder.
  • myhostname: Agar server menyapa dengan nama keren "mail.lks2026.id" (bukan main.localdomain).
  • mynetworks: Izin agar IP Laptop (10.10.10.x) boleh numpang kirim email.
root@mail:~# systemctl restart postfix

3. Config Dovecot (Agar Login Mudah)

A. Lokasi Surat:

root@mail:~# nano /etc/dovecot/conf.d/10-mail.conf
mail_location = maildir:~/Maildir

B. Izin Plaintext (PENTING!):

root@mail:~# nano /etc/dovecot/conf.d/10-auth.conf
disable_plaintext_auth = no
root@mail:~# systemctl restart dovecot

4. Buat User Siswa

root@mail:~# adduser siswa
root@mail:~# maildirmake.dovecot /etc/skel/Maildir
4

Webmail (Opsional)

Lompati jika paket tidak ada.

Catatan Strategi: Jika perintah di bawah error "Unable to locate package", ABAIKAN SAJA. Langsung ke Langkah 5. Nilai inti ada di pengiriman email, bukan tampilan web.

root@mail:~# apt install mariadb-server roundcube roundcube-mysql -y

Jika berhasil: Edit /etc/apache2/conf-available/roundcube.conf (Uncomment Alias), lalu reload apache2.

5

Pembuktian (Laptop Fisik)

Ini adalah poin penilaian utama.

Cek Koneksi Dulu

Pastikan Laptop Fisik sudah aman:

  • IP Laptop: 10.10.10.x (DHCP atau Manual).
  • Gateway Laptop: 10.10.10.1.
  • Ping: ping 172.16.10.30 harus Reply.

Metode A: Telnet (CMD Windows)

Cara paling cepat dan keren di mata Juri.

1. Tes SMTP (Port 25)
C:\> telnet 172.16.10.30 25 220 mail.lks2026.id ESMTP Postfix...
2. Tes POP3 (Port 110)
C:\> telnet 172.16.10.30 110 +OK Dovecot ready.

Jika kedua tulisan hijau itu muncul, ANDA LULUS!

Metode B: Thunderbird / Outlook (Pro)

Jika ada aplikasinya di laptop, gunakan ini untuk kirim email sungguhan.

  • Buka Thunderbird > Add Account.
  • Email: siswa@lks2026.id, Pass: 1.
  • Manual Config:
    • IMAP: 172.16.10.30 (Port 143, SSL: None).
    • SMTP: 172.16.10.30 (Port 25, SSL: None).
    • Username: siswa (Tanpa @domain).
