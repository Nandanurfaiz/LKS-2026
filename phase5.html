DMZ Area

Layanan Surat Elektronik

Membangun Mail Server lengkap (Postfix/Dovecot) dan Webmail (Roundcube) agar email bisa diakses via browser layaknya Gmail.

Target Utama

VM Mail:

  • IP: 172.16.10.30 (VMnet10)
  • Domain: mail.lks2026.id
  • Webmail: http://mail.lks2026.id/roundcube

Metode Pengujian:

  • Backend: Telnet via CMD Laptop.
  • Frontend: Browser (Chrome/Firefox).
  • BARU! Webmail: Kirim/Terima email via GUI.
1

Setup VM Mail

Konfigurasi Dasar.

1. IP Address (Pastikan Benar!)

root@mail:~# nano /etc/network/interfaces
auto ens33 iface ens33 inet static address 172.16.10.30/24 gateway 172.16.10.1 dns-nameservers 172.16.10.1
root@mail:~# hostnamectl set-hostname mail
root@mail:~# systemctl restart networking
2

Update DNS (Router)

Pindah ke VM fw.

1. Tambah Record Mail

root@fw:~# nano /etc/bind/db.lks

Tambahkan 2 baris ini di paling bawah:

@ IN MX 10 mail mail IN A 172.16.10.30

2. Restart Bind9

root@fw:~# systemctl restart bind9
3

Install Postfix & Dovecot

Kembali ke VM mail.

1. Install Paket Inti

root@mail:~# apt install postfix dovecot-imapd dovecot-pop3d -y

2. Config Postfix

root@mail:~# nano /etc/postfix/main.cf

Tambahkan/Edit baris di bagian paling bawah:

home_mailbox = Maildir/ myhostname = mail.lks2026.id mynetworks = 127.0.0.0/8 172.16.10.0/24 10.10.10.0/24
root@mail:~# systemctl restart postfix

3. Config Dovecot

root@mail:~# nano /etc/dovecot/conf.d/10-mail.conf
mail_location = maildir:~/Maildir
root@mail:~# nano /etc/dovecot/conf.d/10-auth.conf
disable_plaintext_auth = no
root@mail:~# systemctl restart dovecot

4. Buat User Siswa

root@mail:~# adduser siswa
root@mail:~# maildirmake.dovecot /etc/skel/Maildir
4

Webmail (Roundcube)

Materi Baru (Wajib!)

Kita akan menginstall Roundcube. Roundcube versi Debian 12 sudah otomatis membuat Alias di Apache, jadi kita tinggal config koneksinya.

1. Install Roundcube & Database

root@mail:~# apt install roundcube roundcube-mysql -y
Saat Instalasi Muncul Pertanyaan:
1. Configure database for roundcube with dbconfig-common? -> YES
2. MySQL application password for roundcube -> [Kosongkan/Enter] (Biar random)

2. Konfigurasi Host (PENTING!)

Di Debian 12, parameter konfigurasi sedikit berubah. Perhatikan baik-baik.

root@mail:~# nano /etc/roundcube/config.inc.php

A. Cari baris $config['imap_host'], ubah menjadi:

$config['imap_host'] = 'localhost';

B. Cari baris $config['smtp_host'], ubah menjadi:

$config['smtp_host'] = 'localhost:25';

*Kita pakai port 25 karena port submission (587) belum diaktifkan.
