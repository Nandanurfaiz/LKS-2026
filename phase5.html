Layanan Surat Elektronik

Kita akan membangun Server Email di zona DMZ agar Laptop Client (Fisik) bisa mengirim dan menerima pesan melalui Webmail.

Target Configuration

VM Info (Mail):

  • Hostname: mail
  • IP Address: 172.16.10.30 (DMZ)
  • Gateway: 172.16.10.1 (Router fw)
  • Domain: mail.lks2026.id

Paket Software:

  • Postfix: SMTP Server (Pengirim Surat).
  • Dovecot: IMAP/POP3 (Kotak Surat).
  • Roundcube: Webmail (Tampilan Web).
1

Setup VM Mail

Clone & Config IP.

1. Cloning

Matikan Debian-Master. Clone menjadi VM baru bernama mail.

PENTING: Network Adapter VM ini tetap VMnet10 (DMZ). Jangan di-Bridge! Yang di-Bridge hanya Router.

2. Konfigurasi IP Static

root@mail:~# nano /etc/network/interfaces
auto ens33 iface ens33 inet static address 172.16.10.30/24 gateway 172.16.10.1 dns-nameservers 172.16.10.1
root@mail:~# hostnamectl set-hostname mail
root@mail:~# systemctl restart networking

PENTING: Jangan lupa masukkan DVD-1/2/3 ke VM Mail dan jalankan apt-cdrom add agar bisa install paket.

2

Update DNS (Wajib!)

Agar "mail.lks2026.id" dikenali.

Pindah ke VM Router (fw) untuk langkah ini!

1. Tambah Record Mail & MX

root@fw:~# nano /etc/bind/db.lks

Tambahkan 2 baris ini di bagian bawah:

@ IN MX 10 mail.lks2026.id. mail IN A 172.16.10.30

2. Restart Bind9

root@fw:~# systemctl restart bind9
3

Install Mail Server

Kembali ke VM mail.

1. Install Paket

root@mail:~# apt install postfix dovecot-imapd dovecot-pop3d -y
Layar Konfigurasi Postfix (Warna Pink/Biru):
  • Type: Internet Site
  • System mail name: lks2026.id (Hapus 'mail.', sisakan domain induk)

2. Konfigurasi Postfix (Maildir)

Agar email tersimpan rapi per file:

root@mail:~# nano /etc/postfix/main.cf

Tambahkan baris ini di paling bawah:

home_mailbox = Maildir/
root@mail:~# systemctl restart postfix

3. Konfigurasi Dovecot

root@mail:~# nano /etc/dovecot/conf.d/10-mail.conf

Cari baris mail_location, ubah menjadi:

mail_location = maildir:~/Maildir
root@mail:~# systemctl restart dovecot

4. Buat User & Folder

Membuat user baru untuk tes email:

root@mail:~# adduser siswa
# Pancing folder Maildir agar terbentuk:
root@mail:~# maildirmake.dovecot /etc/skel/Maildir
4

Webmail (Roundcube)

Akses email via Browser.

1. Install Database & Roundcube

root@mail:~# apt install mariadb-server roundcube roundcube-mysql -y
Layar Konfigurasi dbconfig-common:
  • Configure database for roundcube? Yes
  • Database type: mysql
  • Password for application: (Isi bebas, misal 123)

2. Konfigurasi Apache Alias

Agar bisa diakses via /mail:

root@mail:~# nano /etc/apache2/conf-available/roundcube.conf

Hapus tanda pagar (#) pada baris:

Alias /roundcube /var/lib/roundcube
root@mail:~# systemctl reload apache2

Pengujian (Laptop Client)

Karena Anda menggunakan Strategi B (Client Fisik):

  • Pastikan Laptop terhubung kabel LAN ke PC Server.
  • Pastikan Laptop mendapat IP DHCP (10.10.10.x).
  • Buka Browser di Laptop (Chrome/Edge).
  • Ketik alamat: http://mail.lks2026.id/roundcube
  • Login dengan user: siswa, password: 123 (atau yang Anda buat).

Troubleshooting

Email tidak terkirim / Error 504
Solusi: Cek file /var/log/mail.log. Pastikan di main.cf baris mynetworks mengizinkan IP Client (misal `0.0.0.0/0` atau `10.10.10.0/24`).
Roundcube "Connection to Storage Server Failed"
Solusi: Ini berarti Dovecot mati atau salah config. Cek `systemctl status dovecot`. Pastikan folder `Maildir` ada di home user.
