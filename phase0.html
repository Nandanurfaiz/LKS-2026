Fase 0: Golden Master (HAProxy Only)
Pre-Competition
Efficiency Strategy

Persiapan Paket HAProxy

Mengambil `Debian-Master` untuk disuntik HAProxy via internet sebelum kompetisi dimulai.

Mengapa Hanya HAProxy?

Paket standar seperti Bind9, Postfix, dan Samba biasanya tersedia lengkap di DVD-1 Binary. Menginstallnya secara online di Master tidak memberikan perbedaan signifikan.

Namun, HAProxy seringkali memerlukan library SSL dan Lua versi spesifik yang ada di DVD-2 atau DVD-3. Menyiapkannya sekarang menjamin kelancaran Fase 7 nanti.

1

Cloning Master Polos

Menduplikasi pondasi dari Fase 1.

  1. Pastikan VM Debian-Master (hasil Fase 1) dalam keadaan Power Off.
  2. Klik Kanan > Manage > Clone.
  3. Pilih Full Clone. Beri nama: Golden-Master.
  4. Ubah Network Adapter VM baru ini ke NAT agar bisa meminjam koneksi internet dari Laptop Host.
2

Aktifkan Jalur Internet

Menghubungkan sistem ke repositori pusat Debian.

1. Update Source List

root@master:~# nano /etc/apt/sources.list

Tambahkan baris berikut (Debian 12 Bookworm):

deb http://deb.debian.org/debian bookworm main deb http://security.debian.org/debian-security bookworm-security main

2. Injeksi HAProxy

Pastikan ping ke google.com sukses sebelum perintah ini:

root@master:~# apt update
root@master:~# apt install haproxy -y

Aturan Emas: JANGAN jalankan 'apt clean'!

Kita butuh file .deb tetap tersisa di /var/cache/apt/archives agar cloning berikutnya tidak butuh internet.

3

Segel & Bersihkan

Mengembalikan kondisi Master ke mode "Lomba".

Hapus rujukan internet di source list agar juri tidak menemukan jejak repositori online.

root@master:~# nano /etc/apt/sources.list

# Hapus atau beri tanda pagar (#) kembali pada baris deb http://

root@master:~# poweroff

Golden Master: Injected

Setiap VM yang dikloning dari Golden-Master ini sekarang sudah memiliki paket HAProxy di dalam sistemnya. Anda tinggal mengetik `apt install haproxy` saat lomba nanti tanpa perlu koneksi internet.
