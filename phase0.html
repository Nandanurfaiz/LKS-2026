Fase 0: Golden Master Setup
Pre-Competition
STRATEGI INJEKSI PAKET

Menyiapkan "Golden Master"

Memanfaatkan koneksi internet di awal untuk mengamankan paket HAProxy sebelum kompetisi dimulai.

Mengapa melakukan ini?

Beberapa paket seperti HAProxy atau Roundcube terkadang tidak ditemukan di DVD-1. Dengan menginstallnya saat persiapan (saat internet masih ada), paket tersebut akan masuk ke dalam APT Cache di Master VM. Saat hari-H, meskipun internet dicabut, proses kloning akan membawa paket tersebut ke server tujuan.

1

Koneksi Internet Master

Pastikan Network Adapter Master VM diset ke NAT dan sudah bisa ping google.com.

root@master:~# nano /etc/apt/sources.list

Tambahkan repo resmi Debian Bookworm (pastikan internet PC Host aktif):

deb http://deb.debian.org/debian bookworm main
deb http://security.debian.org/debian-security bookworm-security main
2

Suntik Paket (Online)

Kita install sekarang agar Master VM memiliki semua dependensi yang diperlukan.

root@master:~# apt update
root@master:~# apt install haproxy bind9 postfix dovecot-imapd dovecot-pop3d samba ntp -y

JANGAN jalankan 'apt clean' setelah ini, agar file installer (.deb) tetap tersimpan di folder /var/cache/apt/archives.

3

Lepas Internet (Finalisasi)

Kembalikan Sources List

Hapus kembali link deb http:// agar tidak dicurigai juri. Pastikan hanya deb cdrom: yang aktif.

root@master:~# nano /etc/apt/sources.list

Matikan Mesin

Golden Master siap dikloning ke semua fase berikutnya tanpa butuh internet lagi.

root@master:~# poweroff

Master Siap Digunakan!

Gunakan VM ini sebagai sumber Cloning untuk Fase 2 s/d Fase 10.
